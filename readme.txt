# ExcelToPptTool

A Python tool to convert Excel data into PowerPoint presentations.

## Description

This tool reads data from an Excel file and automatically generates a PowerPoint presentation based on that data. It is useful for quickly converting reports or data tables into presentation slides.

## Features

- Extracts data from Excel sheets.
- Converts data into slides in PowerPoint.
- Allows customization of the input sheet for specific needs.

## Requirements

- Python 3.x
- Required Python libraries:
  - `pandas`
  - `python-pptx`
  - `openpyxl`
